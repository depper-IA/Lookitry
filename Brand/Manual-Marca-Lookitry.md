## 1. IDENTIDAD DE MARCA

### Nombre
**LOOKITRY** (en mayúsculas sostenidas)

> ⚠️ Nunca: "LookiTry", "Looktry", "lookitry" (todo minúsculas)

### Qué es
Lookitry es un **probador virtual con IA** para tiendas de ropa online.

### Propuesta de Valor
"Tu cliente se prueba la ropa antes de comprarla. Sin apps, sin desarrollo."

### Pitch Corto
"Tecnología que permite a los compradores probarse virtualmente la ropa antes de comprarla, reduciendo devoluciones y aumentando ventas."

### Dolor que resuelve
- 😤 El cliente compra algo que no le queda bien
- 📉 Devoluciones costosas para las tiendas
- 🤔 Incertidumbre al comprar moda online

### Solución
- Sin apps por instalar
- Sin desarrollo técnico
- Activa en 10 minutos
- Funciona vía widget en la tienda

### Público Objetivo
- Tiendas de moda online en Latinoamérica
- Marcas que venden por Instagram y WhatsApp
- Mercados: Colombia, México, Argentina, Chile, Perú, Venezuela

## 2. LOGO

### Reglas del Logo
- Usar SOLO la versión oficial proporcionada
- Nunca alterar proporciones
- Nunca cambiar colores del logo
- Mínimo espacio libre: equivalente a la altura de la letra "L"

### Versiones Disponibles
| Versión | Fondo | Uso |
|---------|-------|-----|
| Horizontal | Oscuro | Web, презентации |
| Horizontal | Claro | Fondos claros |
| Icono | Oscuro | Redes sociales, favicon |
| Icono | Claro | Fondos claros |

### Usos NO permitidos
- ❌ Logo sobre fondos muy concurridos
- ❌ Logo distorsionado o rotado
- ❌ Logo con sombra no oficial
- ❌ Logo en escala de grises (a menos que sea versión monocromática oficial)

### Headlines (Títulos)
**Font:** Plus Jakarta Sans
**Fallback:** Segoe UI, Helvetica Neue, Arial

```
font-family: 'Plus Jakarta Sans', 'Segoe UI', sans-serif;
font-weight: 700 (Bold)

**Font:** DM Sans
**Fallback:** Aptos, Segoe UI, sans-serif

```

font-weight: 400 (Regular)

| H3 | 1.5rem (24px) | 600 | Secciones |
| H4 | 1.25rem (20px) | 600 | Cards, destacados |
| Body | 1rem (16px) | 400 | Texto general |
| Small | 0.875rem (14px) | 400 | Texto secundario |

- 🤖 **Innovador** — Somos tecnología de punta
- 😊 **Cercano** — Hablamos como amigo, no como corporación
- 💪 **Confiado** — Sabemos lo que hacemos
- 🇨🇴 **Latino** — Contexto y vibra colombiana

### Lo que SÍ hacemos
- ✅ Hablar directo y sin rodeos
- ✅ Usar "tu" no "usted" (cercanía)
- ✅ Explicar cosas técnicas de forma simple
- ✅ Mostrar confianza sin arrogancia
- ✅ Hacer referencia a cultura latina/colombiana

- ❌ Lenguaje corporativo o formal
- ❌ Anglicismos innecesarios

## 6. ELEMENTOS GRÁFICOS

### Iconografía
**Estilo:** Lucide Icons (outline, stroke 1.5px)

### Bordes y Formas
- Radio de borde: `0.5rem` a `2rem`
- Cards: `border-radius: 1rem`
- Botones: `border-radius: 0.5rem`
- Imágenes de producto: `border-radius: 0.75rem`

### Sombras
```css
/* Sombra sutil para cards */
box-shadow: 0 10px 30px rgba(0, 0, 0, 0.12);

/* Sombra elevada para elementos flotantes */
box-shadow: 0 20px 40px rgba(0, 0, 0, 0.25);
```

### Gradientes
```
/* Gradiente sutil de fondo */
background: linear-gradient(180deg, #1a1a1a 0%, #0a0a0a 100%);

/* Gradiente con color principal */
background: linear-gradient(135deg, #FF5C3A 0%, #ff7a5c 100%);
```

### Espaciado
Sistema de 4px:
```
4px — 8px — 12px — 16px — 24px — 32px — 48px — 64px — 96px
```

### Border Width
```
1px — Líneas sutiles
2px — Elementos de interfaz
3px — Highlights y énfasis
```

## 7. APLICACIONES Y USOS

### Dark Mode (Principal)
El sitio y app usan dark mode por defecto
- Fondo: `#0a0a0a`
- Cards: `#1a1a1a`
- Texto principal: `#FFFFFF`
- Texto secundario: `#BBBBBB`

### Fondos de Imágenes
- Usar fondos oscuros cuando haya foto de producto

## 8. LO QUE NO DEBEMOS HACER

###❌ NO USAR
- Colores neón o fluorescentes
- Más de 3 colores juntos
- Tipografías decorativas para cuerpo de texto
- Mayúsculas sostenidas (solo para el nombre LOOKITRY)
- Italics excesivos
- Sombras de colores raros
- Fondos con patrones muy cargados

###❌ NO DECIR
- "Plataforma integral de soluciones..."
- "Somos pioneros en..."
- "Le ayudamos a optimizar sus procesos..."
- Demasiada jerga técnica

###❌ NO HACER
- Cambiar la propuesta de valor
- Prometer lo que no podemos cumplir
- Usar fotos de stock genéricas
- Copiar estilo de competidores
