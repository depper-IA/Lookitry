## 3. Formato de Contenido

### 🎬 Reels / TikTok
**Estructura:**
1. **HOOK** (0-3 seg): Gancho visual o pregunta que-stop-scroll
2. **PROBLEMA** (3-8 seg): El dolor del cliente
3. **SOLUCIÓN** (8-15 seg): Lookitry como respuesta
4. **DEMO** (15-30 seg): Muestra el probador funcionando
5. **CTA** (30-60 seg): Enlace a lookitry.com

### 📸 Posts / Carruseles
**Estructura:**
1. **Gancho**: Frase que-stop-scroll
2. **Cuerpo**: Beneficio principal + prueba social
3. **CTA**: Enlace + hashtag
4. **Hashtags**: 5-10 relevantes

### 📱 Stories Secuenciales
- 5-7 stories conectadas
- Cada una con un punto del mensaje
- CTA final al enlace

### 📢 Copy para Ads
- Headline: beneficio claro
- Body: problema + solución
- CTA: acción específica
