# Lookitry Brain Vault

Vault de Obsidian del proyecto Lookitry. Configurado como segundo cerebro y fuente de verdad documental del proyecto.

## Estructura

```
Lookitry_Brain_Vault/
├── Cerebro/                    # Fuente de verdad. Documentación maestra, reglas, protocolos.
│   ├── REGLAS_IMPORTANTES.md    # Reglas absolutas — leer siempre primero
│   ├── MAPA_MAESTRO.md          # Índice de navegación
│   ├── Tech_Stack.md            # Pila tecnológica
│   ├── Design.md                # Sistema de diseño
│   ├── Agents.md                # Orquestación de agentes
│   ├── Docs/                    # Documentos complementarios
│   ├── Protocolos/              # Protocolos operativos
│   └── Agentes/                 # Manuales técnicos por agente
├── .obsidian/                   # Configuración de Obsidian (no editar manualmente)
└── (otros directorios de contenido)
```

## Cómo Abrir en Obsidian

1. Abrir Obsidian
2. "Open folder as vault"
3. Seleccionar este directorio (`Lookitry_Brain_Vault/`)

## Regla de Oro

> **El Cerebro es la fuente de verdad. Obsidian es el visor.**

- Leer `Cerebro/REGLAS_IMPORTANTES.md` antes de cualquier acción.
- El contenido del Cerebro prevalece sobre cualquier metadata de Obsidian.
- Para agregar un documento, crear el archivo en `Cerebro/` y actualizar `MAPA_MAESTRO.md`.
