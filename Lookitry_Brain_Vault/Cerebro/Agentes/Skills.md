## Code Sync Checker

**Proposito:** Evitar duplicacion de codigo.

**Regla:** ANTES de crear cualquier funcion/componente/endpoint, BUSCAR si ya existe.

**Ubicacion:** `Cerebro/Skills/code-sync-checker.md`

---

## Brainstorming

**Proposito:** Exploration de user intent y requisitos antes de implementacion.

**Regla:** Usar ANTES de cualquier trabajo creativo - crear features, componentes, o modificar comportamiento.

**Ubicacion:** `.opencode/skills/brainstorming/SKILL.md`

---

## Refactor

**Proposito:** Guia para refactorizar archivos grandes (>600 lineas).

**Regla:** Cuando un archivo supere 600 lineas, extraer componentes/funciones.

**Ubicacion:** `.opencode/skills/refactor/SKILL.md`
