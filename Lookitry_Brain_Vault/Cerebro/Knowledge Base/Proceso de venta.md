# Proceso para contratar Lookitry

1. Registro en https://lookitry.com/checkout
2. Pago del trial de 7 días ($20.000 COP) para explorar el dashboard y configurar el widget
3. Al finalizar el trial, elegir plan BASIC ($180.000/mes) o PRO ($350.000/mes)
4. Pago con Wompi (Colombia) o PayPal (internacional)
5. El widget queda activo inmediatamente después del pago

También se puede contratar directamente sin trial: elegir plan y pagar.
