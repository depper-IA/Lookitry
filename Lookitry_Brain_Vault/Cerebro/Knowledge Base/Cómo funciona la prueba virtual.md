# Cómo funciona la prueba virtual

El cliente de la tienda sube una foto suya, selecciona una prenda del catálogo y la IA genera una imagen realista mostrando cómo le quedaría la ropa. Todo desde el navegador, sin app ni descarga.

El proceso toma menos de 30 segundos.

Funciona con: camisas, camisetas, vestidos, pantalones, faldas, chaquetas, abrigos y accesorios. Funciona mejor con prendas de cuerpo completo.
