# Compatibilidad con plataformas

Lookitry funciona con cualquier plataforma que permita insertar código HTML/JavaScript:

- Shopify
- WooCommerce (con plugin oficial en plan PRO)
- Tiendanube
- Wix
- Squarespace
- Sitio web propio

Si la plataforma permite pegar un script, Lookitry funciona.
