# 🛠️ Superpoderes Técnicos (Skills)

Este es el catalogo de habilidades reales que los agentes utilizan para construir y operar Lookitry. Estas habilidades no son solo guias, son estandares de ejecucion obligatorios.

## 🧠 Planeacion y Estrategia
- [[./brainstorming]]: Fase de ideacion antes de cada tarea. ( Instalar: `npx skills add obra/superpowers@brainstorming -g -y` )
- [[./sequential-thinking]]: Logica profunda para problemas complejos. ( Instalar: `npx skills add mrgoonie/claudekit-skills@sequential-thinking -g -y` )

## 🎨 Diseño y Frontend
- [[./ui-ux-pro-max]]: Manual para interfaces de clase mundial. Implementa estetica premium. ( Instalar: `npx skills add nextlevelbuilder/ui-ux-pro-max-skill@ui-ux-pro-max -g -y` )
- [[./frontend-design]]: Reglas de diseño reactivo y visual. ( Instalar: `npx skills add anthropics/skills@frontend-design -g -y` )

## 🛡️ Calidad y Control
- [[./verification-loop]]: Garantia de que todo funcione antes de entregarlo. ( Instalar: `npx skills add affaan-m/everything-claude-code@verification-loop -g -y` )
- [[./code-reviewer]]: Estandares de limpieza y modularidad de codigo. ( Instalar: `npx skills add google-gemini/gemini-cli@code-reviewer -g -y` )
- [[./requesting-code-review]]: Protocolo para solicitar reviews expertos. ( Instalar: `npx skills add obra/superpowers@requesting-code-review -g -y` )

## 🏢 Habilidades Propias de la IA
- [[./subagent-driven-development]]: Trabajo en equipo para grandes implementaciones. ( Instalar: `npx skills add obra/superpowers@subagent-driven-development -g -y` )
- [[./multi-reviewer-patterns]]: Patrones de multiples revisores. ( Instalar: `npx skills add wshobson/agents@multi-reviewer-patterns -g -y` )
- [[./mcp-builder]]: Creacion de herramientas MCP para integrar servicios. ( Instalar manualmente: ver directorio `~/.agents/skills/mcp-builder/` )
- [[./excalidraw-diagram]]: Generacion de diagramas Excalidraw. ( Instalar: `npx skills add coleam00/excalidraw-diagram-skill@excalidraw-diagram -g -y` )

---

> [!IMPORTANT]
> Los agentes deben invocar estas habilidades al inicio de sus tareas para asegurar consistencia total con el Cerebro.
