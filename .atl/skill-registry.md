## Project Conventions

- **Framework**: Next.js 14 App Router + Express API
- **Database**: Supabase (PostgreSQL + pgvector)
- **Styling**: Tailwind CSS 3.4.0
- **Testing**: Vitest (FE) + Jest (BE) + fast-check
- **AI**: n8n + OpenRouter for Try-On
- **Design**: Plus Jakarta Sans / DM Sans, colors `#FF5C3A` / `#0a0a0a`
