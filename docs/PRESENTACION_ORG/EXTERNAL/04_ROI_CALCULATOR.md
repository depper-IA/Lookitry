# Lookitry - ROI Calculator

**Calculadora de retorno de inversión para retailers**

---

## Qué Resuelve Lookitry

El **30% de las compras de moda online son devueltas**. La causa principal: el cliente no sabe cómo le va a quedar la ropa.

Cada devolución le cuesta al retailer entre **15-30% del valor del pedido** en logística (envío, manejo, reestiba).

Lookitry reduce ese problema dejando que el cliente **se pruebe virtualmente antes de comprar**.

---

## Modelo de Impacto

### Antes de Lookitry
```
Cliente ve prenda → Compra → No le queda bien → Devuelve

Costo para el retailer por devolución:
- Envío de ida: $XX.XXX
- Manejo en warehouse: $X.XXX
- Reestiba/reempaque: $X.XXX
- Envío de vuelta: $XX.XXX
- Pérdida potencial del item (dañado, usado): $XX.XXX
─────────────────────────────────────
Total: ~15-30% del valor del pedido
```

### Con Lookitry
```
Cliente ve prenda → Se prueba virtualmente → Decide → Compra (o no)

Si no le queda bien, no compra:
- Evita la devolución ANTES de que ocurra
- Solo "pierde" el tiempo de probarse (12 segundos)

Si le queda bien, compra:
- Menos chance de devolver después
- Más confianza → menos devoluciones
```

## Factores que Amplifican el ROI

### 1. Aumento de Conversión
El cliente que se prueba virtualmente tiene **más confianza** para comprar.
- Dato de la industria: +5-15% en conversión en fichas con try-on virtual
-Impacto: Más ventas por el mismo tráfico

### 2. Reducción de Costos de Atención
Menos devoluciones = menos tickets de soporte, menos logística, menos estrés operativo.

### 3. Mejor NPS
Clientes satisfechos que no se arrepienten compran de nuevo.
- lifetime value aumenta
- word-of-mouth aumenta
